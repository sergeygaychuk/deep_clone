# DeepCopy

![alt tag](http://i.qkme.me/356wcq.jpg)

## Installation

Add this line to your application's Gemfile:

    gem 'deep_copy'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install deep_copy

## Usage

    require 'deep_copy'

    {a: 1, b: 2, c: {d: 3}, e: [100, 101, 102]}.__deep_copy__
